# ZK-Ballot: Anonymous Voting System

A decentralized, privacy-preserving voting system built with commitment schemes and Merkle tree verification for secure governance.

## Deployed Contract

**Contract Address:** `0x2324BF3d2313532D1D1Ce946947b80661826a49E`  
**Network:** Sepolia Testnet  
**Etherscan:** https://sepolia.etherscan.io/address/0x2324BF3d2313532D1D1Ce946947b80661826a49E

## Quick Start

```bash
npm install
npm run compile
npm test
npm run deploy:testnet
```

## Technical Features

- Commitment-reveal voting scheme
- Merkle tree eligibility verification
- Time-locked voting phases  
- Weighted voting support
- Role-based access control

See full documentation in the repository.
